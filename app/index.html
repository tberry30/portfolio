About

Hello, world! I'm Tim.


I'm a full-stack web developer from Salt Lake City, Utah, with a background in inventory management, the outdoor industry, and music production.

Since I started dabbling in the web several years ago, I have found a passion for developing responsive, clean websites that offer a great user experience. As a child, did you ever take things apart just to see what made them work? That was me. I began by following this same curiosity that I had as a child. I would dissect source code, look at all the parts, and try to figure out how it all worked together to make a functional site. As I started to understand all the pieces, I would make changes to see their impact. This is how I learned the basics of development.

When I decided to make this my career, I attended DevMountain, where I gained advanced skills in full-stack development, focusing on Angular and Node. Now as a professional web developer, I have built a skill set of many web technologies. If you're in need of a website, or looking to hire, please get in touch here.
